Introduction

This is the official repository for the Haskell implementation of the ABCI server and SDK for developing applications backed by the Tendermint replication engine.

This site contains a tutorial, installation instructions as well as haddocks documentation below.

You can read more about Tendermint and the ABCI specs in the tendermint documentation hosted on their website.
