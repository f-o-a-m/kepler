Introduction

This is the official repository for the Haskell implementation of the ABCI server and SDK for developing applications backed by the Tendermint replication engine.

This site contains a tutorial as well as haddocks documentation below.

You can read more about Tendermint and the ABCI specs in the documentation hosted on their website.
